# Songfy Application

Songfy is a web application designed for music streaming and management. It provides users with a seamless experience to explore their music library, create playlists, and access various statistics related to their listening habits.

## Project Structure
```
songfy
├── src/
│   ├── index.html
│   ├── assets/
│   ├── styles/
│   ├── scripts
│   └── musics/
│
├── docs/
│
├── assets/
│   ├── data.json
│   └── image/
│
└── README.md
```

## Features
- **Responsive Design**: Utilizes Bootstrap 5 for a fluid layout that adapts to different screen sizes.
- **Navigation**: Includes a navbar with links to key sections: Início, Biblioteca, Playlists, and Estatísticas.
- **Sidebar Navigation**: A left sidebar for quick access to various features with icons for easy identification.
- **Music Player**: A fixed player at the bottom of the page for continuous playback.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/bybenb/showcase-2026.git
   ```
2. Navigate to the project directory:
   ```
   cd showcase-2026/2-songfy/
   ```
3. Open `src/index.html` in your web browser to view the application.

## Usage
- Use the navbar to navigate between different sections of the application.
- Access your music library and create playlists from the sidebar.
- Monitor your listening statistics in the Estatísticas section.
